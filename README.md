 Atelier Lívia Fontenelle | E-commerce Premium
Uma plataforma de e-commerce de joias de alto padrão desenvolvida com Next.js 15, focada em luxo minimalista, alta performance e uma experiência de usuário fluida. Este projeto foi criado para apresentar coleções exclusivas com uma presença digital sofisticada.

- Funcionalidades Principais

   Roteamento Dinâmico de Produtos: Utiliza o sistema de rotas assíncronas do Next.js 15 para gerar páginas únicas e de alta performance para cada joia da coleção.

   Sistema de Carrinho em Tempo Real: Integração com React Context API para gerenciar um carrinho global, permitindo que o usuário adicione ou remova itens e veja atualizações de preço instantaneamente em qualquer página.

   Filtro de Catálogo Inteligente: Categorização interativa (Anéis, Colares, Brincos, Relógios) que renderiza a vitrine em tempo real sem recarregar a página.

   UI/UX de Luxo: Design pensado com abordagem "Mobile-First" utilizando Tailwind CSS, apresentando transições suaves, tipografia serifada e uma paleta de cores refinada em dourado e preto.

   Gerenciamento de Dados Centralizado: Arquitetura baseada em uma "fonte única da verdade" para todas as informações dos produtos, garantindo consistência e facilidade de manutenção.

- Tecnologias Utilizadas

    Framework: Next.js 15 (App Router)

    Linguagem: TypeScript (para manipulação de dados com tipagem segura)

    Estilização: Tailwind CSS (CSS utilitário)
    
    Gerenciamento de Estado: React Context API (memória global do carrinho)
    
    Ativos Visuais: Fotografia de joias em alta resolução e ícones minimalistas.

- Como Executar o Projeto
      Clone o repositório:
      
      git clone https://github.com/paulovitor23/Jowelry_store.git
      npm install
      npm run dev
      
     Navegue para http://localhost:3000 para visualizar o atelier.

- Filosofia de Design

    O projeto segue o princípio "Menos é Mais". Através do uso generoso de espaços em branco (padding), grids consistentes e micro-interações sutis, a interface mimetiza a elegância silenciosa de uma boutique física de alta joalheria.
